# 目录结构

```text
├── mod.rs                # 模块声明和公共接口
├── types.rs              # 数据结构定义
├── config.rs             # 配置常量和线程池
├── cache.rs              # 缓存相关功能
├── preprocessing.rs      # 图片预处理和分块
├── chunk_processing.rs   # 单个chunk处理
├── commands.rs           # Tauri命令函数
└── utils.rs              # 工具函数
```
